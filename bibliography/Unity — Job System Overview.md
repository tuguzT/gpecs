# Unity — Job System Overview

The Job System is a Unity framework for writing multithreaded code so an application can use all available CPU cores.

It is commonly used together with the [Burst](Unity%20DOTS%20—%20Burst%20Compiler%20Documentation.md) compiler,
but also could be used with with Unity’s [ECS](Unity%20DOTS%20—%20Entities%20Documentation.md).

# Unity DOTS — Burst Compiler Documentation

Burst is a compiler that translates a subset of C# used for high-performance code
from .NET IL into optimized native CPU code using LLVM.

It was designed to be used with Unity’s [Job System](Unity%20—%20Job%20System%20Overview.md):
jobs and other compatible code can be compiled with Burst, compile artifacts then used to reduce CPU time for CPU-bound workloads.
